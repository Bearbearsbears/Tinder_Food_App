# Tinder_Food_App
Project for CSC 3380

Step 1. Download Zip file by clicking green code button in the main screen.

Step 2. Open Windows PowerShell

Step 3. Extract Zip File and move the inner folder from the outer folder to the directory shown when you open Windows PowerShell, for example: C:

Step 4. Move into the Tinder_Food_App-MVP_Milestone folder Using: cd .\Tinder_Food_App-MVP_Milestone\ (Make sure your pathway is correct, if you put the folder in the C Drive make sure the bash command is 
C:\> cd .\Tinder_Food_App-MVP_Milestone\)

Step 5. Install NPM, with the command: npm install --legacy-peer-deps, then this: npm audit fix, then this: npm install firebase, this this: npm install react-router-dom

Step 6. Run the command: npm run dev and then wait till it shows ROLLDOWN-VITE, next enter the letter o, then press enter. This opens a browser to the webapp.
